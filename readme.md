# Mean Exemplo

## Pre-Requisitos

* MongoDB
* Express
* NodeJS
* Bower
* Gulp

## Subir Servidor

```
$ nodemon api-server.js
```

## Subir Banco de dados

```
$ mongod
```

## Front-end

### Baixar as dependências

```
$ bower install
```

## Acessar Aplicação

http://seu-ip:3000

## Acessar API

http://seu-ip:3000/api/aluno
